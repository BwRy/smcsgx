Rules for writing SFE input for Evaluator/Sender

1. Lines starting with '#' will be ignored

2. The first valid line should be
	SFE - for SFE mil
	SFEUC - for SFE uc
	SFEDJ - for SFE dijk

3. Every input should hold one line if there are multiple inputs

4. Every member should hold one line if the input is a structure

5. Have to enumerate all the values within the line if the input is an array

6. Use CSV format for the array case mentioned above
